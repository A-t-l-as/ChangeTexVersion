# ChangeTexVersion

## ENG:

A small program for changing the version of `.TEX` files.<br>
This program is used to change the texture version in every header throughout the entire file.<br>

## How to use:
`ChangeTexVersionValue.exe <The searched value of the texture version (1 or 2)> <The value you want to change to> <Path to the source file> <Path to the target file>`

Compiling the latest version requires the compiled static library `KsUtils`.<br>
[Link](https://github.com/A-t-l-as/LibKsUtils)<br>

---

## PL:

Mały program do zmiany wersji plików `.TEX`.<br>
Ten program służy do zamiany wersji textury w każdym jej nagłówku w całym pliku.<br>

## Sposób użycia:
`ChangeTexVersionValue.exe <Wyszukiwana wartość wersji tekstury (1 lub 2)> <Wartość na która chcesz zmienić> <Ścieżka do pliku źródłowego> <Ścieżka do pliku docelowego>`

Kompilacja najnowszej wersji wymaga skompilowanej statycznej biblioteki `KsUtils`.<br>
[Link](https://github.com/A-t-l-as/LibKsUtils)<br>
