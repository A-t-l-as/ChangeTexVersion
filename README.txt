PL:

Ten program służy do zamiany wersji textury w każdym jej nagłówku w całym pliku.

Sposób użycia:
ChangeTexVersionValue.exe <Wyszukiwana wartość wersji tekstury (1 lub 2)> <Wartość na która chcesz zmienić> <Ścieżka do pliku źródłowego> <Ścieżka do pliku docelowego>

---------------------------------------------------------------------------------------------------------------------------------------------------------

ENG:

This program is used to change the texture version in every header throughout the entire file.

How to use:
ChangeTexVersionValue.exe <The searched value of the texture version (1 or 2)> <The value you want to change to> <Path to the source file> <Path to the target file>
